8.2
border-callapse The border-calapse attribute is used to choose between sperate or callapsed border models. To apply use the border-callapse:type; where type is either seperate or callapse
border-spacing The border-spacing attribute is used to set the spacing between borders. To apply this use the border-spacing: value; Where value is in CSS units of measure.
rowspan Rowspan is used to span a table cell across multiple rows. to apply this use the attribute rowspan="rows". Where rows are the amount of rows to take up.
colspan colspan is used to span a table cell over multiple columns. To apply this use the attribute colspan="cols" . Where cols is the number of columns to take up.
8.3
caption-side The caption-side is used to define where the caption is placed. To apply the cation-side use caption-side:position;. Where position is either top or bottom.
id the attribute id is used to identify a column/row. To apply this use the id="name". Where name is the name you give the column/row.
class the attribute class is used to identify a column/row. To apply this use the class="name". Where name is the name you give the column/row.
Vertical-align The vertical-align attribute is uses to align cell text vertically. to apply this attribute use vertical-align: type;. Where type can be middle, baseline, %, sub, super, top, text-top, middle, bottom, text-bottom, initial, or inherit.
display The display attribute is used to dispaly tables certain ways. to apply this attribute display:type;. Where type can be table, table-inline, table-row, table-row-group, table-header-group, table-footer-group, table-column, table-column-group, table-cell,or table-caption.
