Unit 9 Vocab
9.1
Web Form
Web Forms allow users to enter data that can be saved and processed, and is a common way to accept a users input
Controls
Controls allow a user to interact with a form.
Widgets
Widgets is another name for Controls
Data field
Data fields store the data values supplied by a user
Input boxes
Input boxes are used to insert text and numeric values
Option Buttons/Radio buttons
Option Buttons/Radio buttons are used to select data values from a predefined set of options
Selection lists
Selection lists are used to select data values from an extensive list of options
Check boxes
Check boxes are used to select data vlaues limited to two possibilities, such as "yes" or "no"
Text area boxes
Text area boxes are used to enter text strings that may include several lines of content
Spin boxes
Spin boxes are used enter integer values confined to specified range
Slider controls
Slider controls are used to enter numeric values confirmed to a specified range
Calendar controls
Calendar controls are used to select date and time values
Color pickers
Color pickers are used to choose color values
9.2
Virtual Keyboards
Virtual Keyboards are software representations of a physical device
9.4
Spinner Control
Spinner Control display an up or down arrow to increase or decrease the field value by a set amount
Range Slider
Range Sliders are a slider control limited to a numeric field to range of possible values
9.5
Form buttons
Form buttons are a type of form control that performs an action
Command buttons
A Command button runs a program that affects the content of a page or the actions of a browser
Submit buttons
Submit buttons submit a form to the server for processing when clicked
Reset buttons
Reset buttons reset a form, changing all fields ti thier default values and deleting any field vlaues that a user has entered
Validation
Validation is the process of ensuring that a user has supplied valid data
Server-side validation
This validation occurs on the web server(after you submit)
Client-side validation
This Validation occurs in the user's browser(Before you submit)
Regular expression or regex
Regular expression or regex is a concise desription of a character pattern
Inline validation
Inline validation is the technique of immediate data validation and reporting of errors
focus
focus is the state in which an element has been clicked by the use, making it the active control on the form
